# /research-trending

Find trending topics and timely content opportunities for AnswerCare AI.

## Process

1. **Check for seasonal opportunities:**
   - Current month → which trades have peak call season?
   - Spring: HVAC (AC), pest control, roofing, landscaping
   - Summer: HVAC (AC emergencies), pest control
   - Fall: HVAC (heat), pest control (rodents), roofing (pre-winter)
   - Winter: Plumbing (frozen pipes), HVAC (heat), locksmiths

2. **Search for trending industry news:**
   - Web search: "ai phone answering for contractors 2025"
   - Web search: "answering service for trades news"
   - Web search: "home services industry trends 2025"

3. **Check Google Trends** (via web search):
   - "plumber answering service" — trending up or down?
   - "hvac answering service" — seasonal spike visible?

4. **Identify timely content angles:**
   - Industry statistics just released (cite and write about)
   - Competitor announcements (new pricing, new features = comparison opportunity)
   - Seasonal service demand spikes = timely landing page or blog post

5. **Output trending opportunities:**
   ```
   SEASONAL (publish now for upcoming peak):
   - [Opportunity] → [Recommended content] → Priority: High/Medium
   
   NEWS-BASED:
   - [Topic] → [Angle] → Source: [URL]
   
   EVERGREEN UPDATE (refresh existing content):
   - [Page] → [What to update]
   ```

Save to `research/trending-[date].md`.
